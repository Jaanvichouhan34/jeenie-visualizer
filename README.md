## 🌟 Overview
Jeenie OS is an immersive 3D landing page that pushes the boundaries of web-based product storytelling. Using **React Three Fiber** and **GSAP ScrollTrigger**, the project features a "Deconstructed" state where the robot model physically explodes into parts as the user scrolls, revealing its internal architecture with cinematic precision.

🔗 **Live Demo:** [Click Here](https://jeenie-visualizer-n81f.vercel.app/)  
📦 **GitHub Repo:** [Click Here](https://github.com/Jaanvichouhan34/jeenie-visualizer)

## ✨ Key Features
- **Exploded View (Deconstruction)**: Scroll-synced GSAP animation that separates the 3D model into technical layers.
- **Real-Time HUD**: Interactive control panel to modify accent colors and emissive lighting states.
- **Intelligent Tracking**: The model's head dynamically follows mouse movement for a lifelike experience.
- **Post-Processing Pipeline**: Professional-grade Bloom, Noise, and Vignette effects for a sci-fi atmosphere.
- **Interactive Hotspots**: Technical data points attached directly to the 3D geometry.

## 🛠️ Tech Stack
- **3D Engine**: Three.js via React Three Fiber
- **Helper Library**: React Three Drei (HTML portals, Centers, Environments)
- **Animation**: GSAP (GreenSock) & ScrollTrigger
- **Post-Processing**: @react-three/postprocessing
- **Build Tool**: Vite (Framework: React)

## 📋 Technical Details
- **Model**: Custom GLB character with optimized PBR textures.
- **Materials**: Dynamically updated emissive and rim-light properties via React state.
- **Responsive**: Sticky container logic ensures the 3D canvas stays fixed while text layers scroll.
